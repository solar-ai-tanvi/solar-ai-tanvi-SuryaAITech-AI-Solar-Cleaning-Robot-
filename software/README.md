# 💻 Software Documentation

This folder contains all software-related documentation for the AI & IoT Enabled Autonomous Solar Panel Cleaning Robot.

---

# Software Architecture

The software is divided into two independent systems:

## Raspberry Pi

Responsible for:

- AI Vision
- Computer Vision
- Image Processing
- Decision Making
- Communication with ESP32

---

## ESP32

Responsible for:

- Motor Control
- Sensor Monitoring
- Navigation
- Cleaning Mechanism
- Battery Monitoring
