# 🔧 Hardware Documentation

This folder contains complete hardware information of the AI & IoT Enabled Autonomous Solar Panel Cleaning Robot.

---

# Hardware Architecture

The robot consists of two processing units:

- Raspberry Pi 5 (AI Vision Processing)
- ESP32 (Robot Control)

Both controllers communicate using UART communication.

---

# Major Hardware Modules

- AI Vision Module
- Robot Control Module
- Navigation Module
- Cleaning Module
- Power Module
- Communication Module
